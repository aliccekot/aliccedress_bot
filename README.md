# aliccedress_bot (Котова Алиса)

##  Описание проекта
Разрабатываемое приложение представляет собой Telegram-бота, предназначенного для автоматизированного подбора одежды на основе пользовательских предпочтений. Основная идея заключается в создании доступного цифрового инструмента, позволяющего пользователю быстро получать готовые образы без необходимости обращения к специалистам.

---

##  Функционал

Пользователь последовательно выбирает параметры, на основе которых формируется рекомендация^

-  **Цветовые предпочтения** — выбор комбинации из трёх цветов  
-  **Стиль одежды** — классический, спортивный, повседневный, вечерний и др.

На основе выбранных параметров система анализирует доступные варианты и предлагает пользователю несколько готовых образов в виде изображений.
